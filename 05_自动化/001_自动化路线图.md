## 技术路线（从简到繁）

### 第一阶段：Script 脚本
- 使用 AutoCAD `SCR` 脚本文件
- 批量执行重复命令
- 示例：批量图层整理、批量打印

### 第二阶段：LISP 程序
- 使用 AutoLISP / Visual LISP
- 较复杂的逻辑判断和循环
- 示例：自动标注、自动门窗编号

### 第三阶段：DXF 解析
- 直接读取/写入 DXF 文本文件
- OpenClaw 可以解析 DXF 结构
- 示例：识别图层、提取尺寸数据

### 第四阶段：外部库集成
- ODA (Open Design Alliance) SDK
- LibreCAD DXF 库
- python-pptx / openpyxl 对接办公软件

### 第五阶段：AI 辅助
- 基于知识库的规则推理
- 图像识别辅助（现场照片→草图）
- 规范检查自动化

### DXF 文件结构基础
```
DXF 文件结构：
  HEADER 段    —— 系统变量
  CLASSES 段   —— 应用类
  TABLES 段    —— 图层、线型、文字样式等
  BLOCKS 段    —— 块定义
  ENTITIES 段  —— 图形实体
  OBJECTS 段   —— 非图形对象
```
