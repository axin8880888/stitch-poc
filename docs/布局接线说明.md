# 布局接线说明

## 首页布局
- `activity_stitch_entry.xml`
- 作用：接片入口壳

## 预览页布局
- `activity_stitch_preview.xml`
- 作用：显示结果字段、裁切、导出按钮

## 预览页文案接线
- `StitchPreviewActivity.renderFields()`
- 输出固定字段列表

## 裁切按钮接线
- `StitchPreviewActivity.onCropClicked()`
- 点击后更新 `PocResultStore` 中的裁切状态

## 原则
- 页面极简
- 文案克制
- 结果可检查
- 按钮少而明确
