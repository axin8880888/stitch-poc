# 预览页完整字段样式模板

## 目标
让预览页信息展示克制、固定、可检查。

## 标题区
- `接片预览`

## 主信息区
```text
title: 接片原型结果
frameCount: 3
status: ready
qualityScore: 0.92
cropMode: pure
cropStatus: enabled:auto-pure-crop
cropBounds: auto-pure-crop
exportStatus: idle
blendEnabled: false
exposureCompensationEnabled: false
```

## 警告区
```text
warning: 帧数过少，建议至少 2 帧
warning: blending默认关闭
warning: exposureCompensator默认关闭
```

## 备注区
```text
note: OpenCV真实接口骨架已接入
note: 默认关闭 blending
note: 默认关闭曝光补偿
```

## 按钮区
- 返回
- 裁切
- 导出

## 视觉原则
- 黑底
- 白字
- 少间距
- 等宽字体展示字段
- 不加多余装饰
