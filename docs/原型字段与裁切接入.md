# 原型字段与裁切接入

## 预览页字段固定化
预览页固定输出：
- title
- frameCount
- status
- qualityScore
- cropMode
- cropStatus
- cropBounds
- blendEnabled
- exposureCompensationEnabled
- warnings
- notes

## 裁切状态接入
裁切状态由结果对象携带：
- cropMode
- cropStatus
- cropBoundsSummary

## 真实 OpenCV native 接口占位
引入 `OpenCvNativeBridge` 作为 JNI 占位层，后续接真实 C++ OpenCV 逻辑。

## 当前原则
- 预览页字段固定且克制
- 裁切状态可被直接检查
- native 接口先占位，后实现
