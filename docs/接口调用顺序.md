# 接口调用顺序

```text
initNative()
→ setBlenderEnabled(false)
→ setExposureCompensatorEnabled(false)
→ prepareNative(frameCount)
→ pushFrameNative(...) x N
→ stitchNative()
→ suggestCropBoundsNative(...)
→ applyPureCropNative(...)
→ writeResultNative(...)
→ releaseNative()
```

## 说明
- Kotlin 负责编排
- JNI 负责真正拼接
- UI 只读结果，不直接碰拼接细节
