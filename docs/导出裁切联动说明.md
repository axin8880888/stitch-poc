# 导出裁切联动说明

## 目标
让预览页的裁切与导出按钮都真正接到原型状态流。

## 裁切按钮
- `StitchPreviewActivity.onCropClicked()`
- 更新：
  - `cropStatus = manual-adjust-requested`
  - `exportStatus = pending`

## 导出按钮
- `StitchPreviewActivity.onExportClicked()`
- 更新：
  - `exportStatus = exporting`

## 入口页导出
- `StitchEntryActivity.onExport()`
- 调用 `ExportManager.saveToFolder(...)`

## 导出默认路径
- `/storage/emulated/0/Download/篮筐整改`

## 原则
- 裁切与导出都必须可回写状态
- 预览页字段必须能马上反映当前状态
- 先确保成功，再提速开发
