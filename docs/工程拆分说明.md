# 工程拆分说明

## 模块划分

### 1. app
- 光边界内入口壳
- 仅负责跳转接片流程

### 2. stitch-core
- OpenCV 拼接核心适配
- 特征提取、匹配、对齐、warp
- 默认关闭 blending 和 exposure compensator

### 3. stitch-ui
- 接片状态页
- 预览页
- 裁切页

### 4. stitch-export
- 保存、分享、相册导出

### 5. stitch-guard
- 主链隔离验证
- 3× / 1.5× / 10× / RAW 保护

## 最小可运行验证路径
1. 入口页可打开。
2. 模拟或接入 2-3 张重叠帧。
3. OpenCV 输出一张拼接图。
4. 预览后做纯裁切。
5. 导出到目录。

## 关键默认值
- blendEnabled = false
- exposureCompensationEnabled = false
- cropMode = pure
- aeLockEnabled = true
- awbLockEnabled = true
