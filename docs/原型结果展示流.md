# 原型结果展示流

## 目标
把伪帧拼接结果输出成一个可直接检查的页面数据流。

## 流程
1. `StitchEntryActivity.onCreate()` 初始化原型。
2. `PocPipeline.runFakeStitch()` 执行伪拼接。
3. `PocResultStore.save()` 保存当前结果。
4. `StitchPreviewActivity.showResult()` 读取并输出文本。
5. 页面显示帧数、状态、裁切模式、blending 开关、曝光补偿开关、备注。

## 当前输出字段
- title
- frameCount
- status
- cropMode
- blendEnabled
- exposureCompensationEnabled
- notes

## 说明
这条流是原型检查流，不是最终 Android UI 实现。
