# 第八阶段：每周训练

**目标：形成持续成长机制——每天练习，每周实战，每月复盘。**

---

## 每日训练计划

每天选择以下内容练习（轮换）：

### 周一：命令练习
- 随机选择 5-8 个命令，每个画 5 次
- 重点练习不熟悉的命令

### 周二：图纸分析
- 找一张真实 CAD 图纸
- 分析图层结构
- 分析标注方式
- 分析图面表达

### 周三：图纸修改
- 对现有图纸进行修改
- 加入新元素
- 调整布局
- 修改标注

### 周四：完整绘图
- 画一张完整的图（平面/立面/节点）
- 从无到有，按规范完成

### 周五：规范学习
- 学习一个新的规范点
- 或复习已学的内容
- 记录到知识库

### 周六：自由练习
- 补做本周没完成的
- 或者针对弱项加强
- 或者研究新工具/新技巧

### 周日：休息 / 总结
- 周末不强制练习
- 如有心情可以看看设计案例

## 每周项目任务

### 第1个项目：美容院门头
- 任务：绘制美容院外观门头设计图
- 包含：正立面图、门头尺寸、招牌位置、材料标注

### 第2个项目：住宅改造 - 平面布置
- 任务：绘制住宅改造平面图
- 包含：拆砌墙标记、家具布置、尺寸标注

### 第3个项目：住宅改造 - 天花/地面
- 任务：绘制天花图 + 地面铺装图
- 包含：吊顶标高、灯具位置、地砖铺装方向

### 第4个项目：住宅改造 - 立面/节点
- 任务：绘制 4 个方向立面图 + 关键节点
- 包含：电视背景墙、厨房、卫生间、柜体

### 后续项目
（按项目清单轮换）

## 月度复盘

每到月底，回顾：
1. 本月完成了几个项目？
2. 每个项目的问题总结
3. 这个月学到了什么新东西？
4. 有哪些规范需要更新？
5. 下个月重点提升什么？
6. 更新 MEMORY.md 记录学习进度
