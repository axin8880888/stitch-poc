## 七、CAD图块使用建议

### 7.1 图块图层
```
家具图块 → P-FURN 图层（颜色 6 紫色）
   洁具 → P-PLMB 图层（颜色 4 青色）
   灯具 → P-ELEC 图层（颜色 1 红色）
```

### 7.2 图块比例
```
1:50 图纸: 图块插入比例 = 1.0
1:100 图纸: 图块插入比例 = 2.0（AI缩放）
```

### 7.3 家具间距
```
沙发↔茶几: 300~400mm
餐桌↔墙: ≥900mm
床↔墙(主过道): ≥800mm
淋浴房↔马桶: ≥200mm
厨房操作区: 台面深度600mm
```
