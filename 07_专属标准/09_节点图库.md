# 09_节点图库

> Guoxin CAD Standard — 常用构造节点索引
> 目的：画施工图大样时参考标准做法

---

## 一、吊顶节点

### 1.1 纸面石膏板吊顶（最常用）
```
┬────┬  龙骨间距300~400
│    │
│ ┌──┤  自攻螺钉间距≤200
│ │  │
│ │  │  9.5mm或12mm纸面石膏板
└─┴──┘
```

**构造层次（从下到上）：**
1. 乳胶漆面层
2. 腻子找平
3. 纸面石膏板 (9.5/12mm)
4. 自攻螺钉固定 (间距≤200mm)
5. 轻钢龙骨 (主龙骨@900，次龙骨@300~400)
6. 吊筋 (直径≥8mm，@≤1200mm)

> 💡 轻钢龙骨吊顶完成厚度：**150~200mm**（含吊筋）

### 1.2 石膏板跌级吊顶
```
                   ────── 上层
                  │
        ─────────┘  灯槽
       │
       │           ────── 下层
       └──────────
```
**尺寸：**
- 跌级高差：**50~150mm**
- 灯槽宽：**100~200mm**
- 阴角处需加**防裂纸带**

### 1.3 铝扣板吊顶（厨房/卫生间）
- 板材：**300×300mm 或 600×600mm**
- 龙骨：三角龙骨，间距与板宽一致
- 边角线：L型收边
- 完成高度：距原顶**≥150mm**（含灯具）

## 二、地面节点

### 2.1 地砖铺贴
```
    地砖 (10~12mm)
    水泥砂浆结合层 (20~30mm)
    水泥砂浆找平层 (20~30mm)
    ─────────────────
    楼板 (结构)
```
**完成厚度：50~70mm**

### 2.2 木地板（悬浮式）
```
    木地板 (12~15mm)
    防潮垫 (2~3mm)
    自流平 (3~5mm)
    ─────────────────
    楼板 (结构)
```
**完成厚度：17~23mm**

### 2.3 石材地面
```
    石材 (18~20mm)
    水泥砂浆结合层 (30~40mm)
    水泥砂浆找平层 (20~30mm)
    ─────────────────
    楼板 (结构)
```
**完成厚度：70~90mm**

### 2.4 地暖地面
```
    面层 (地砖/木地板)
    水泥砂浆找平 (30~40mm)
    地暖管 (填充层：50~60mm)
    绝热层 (20mm挤塑板)
    ─────────────────
    楼板 (结构)
```
**完成厚度：100~120mm**

---

## 三、墙面节点

### 3.1 墙面涂料
```
    乳胶漆面漆 ×2
    乳胶漆底漆 ×1
    腻子找平 (2~3遍)
    界面剂
    ─────────────────
    墙体 (基层)
```

### 3.2 墙面瓷砖
```
    瓷砖 (8~10mm)
    瓷砖粘结剂 (3~5mm)
    防水层 (卫生间必须)
    水泥砂浆找平 (15~20mm)
    ─────────────────
    墙体 (基层)
```
**完成厚度：25~35mm**

### 3.3 木饰面（干挂）
```
    木饰面板
    结构胶 / 干挂件
    木工板基层 (12~15mm)
    木龙骨 (20×40mm @400)
    ─────────────────
    墙体 (基层)
```
**完成厚度：50~80mm**

### 3.4 轻钢龙骨隔墙
```
     ───┬───┬───┬───    龙骨 @400
        │   │   │
     ───┴───┴───┴───
```
**构造：**
- 龙骨：**75mm或100mm轻钢龙骨**
- 两侧各一层或两层**纸面石膏板**
- 中间可填**岩棉/玻璃棉**（隔音）
- **总厚度：** 100~125mm（75龙骨+双12板）

## 四、门窗节点

### 4.1 门套剖切
```
     ┌─────┐   门套线
     │     │
     │      ─── 门扇
     │     │
     └─────┘
```
**尺寸：**
- 门套线宽：**50~60mm**
- 门套基层：**15mm细木工板**
- 门扇厚度：**40~50mm**

### 4.2 窗套节点
```
     ┌─────┐    窗套
     │     │
     │ ═══ │    玻璃
     │     │
     └─────┘
```
**防水要点：**
- 窗框与墙体间用**发泡胶**填充
- 外侧打**耐候密封胶**
- 内侧**窗台板**（石材/木材）
- 窗台板出墙 **20~30mm**

## 五、收口节点

### 5.1 地砖↔木地板过渡
```
  地砖 │  木地板
       ───
       扣条 (T型/L型)
```
**常用扣条：**
- 高低差≤3mm → 平扣条
- 高低差3~8mm → 收边条
- 高低差>8mm → 斜坡过渡

### 5.2 墙面↔吊顶阴角
```
  墙面 ───
       │
       阴角线 / 石膏线
       │
  吊顶 ───
```
**处理方式：**
- 常规：腻子刮弧（圆角R=20mm）
- 装饰：石膏阴角线（尺寸30/50/80mm）

### 5.3 踢脚线节点
```
  ─── 墙面
  │
  │ ─── 踢脚线 (高80~150mm)
  │
  ─── 地面
```
**踢脚线高度：**
- 现代简约：**80~100mm**
- 欧式/新中式：**120~150mm**

## 六、防水节点（卫生间）

```
     ─────────────────  顶面
     │
     │  墙面防水层上翻 ≥1800mm
     │
     ─────────────────  地面
     地面防水层
     水泥砂浆找平层（坡度≥1%）
     地漏处坡度≥2%
```

**施工顺序：**
```
1. 基层清理 → 2. 管根加强 → 3. 第一遍防水
4. 第二遍防水（十字交叉）→ 5. 闭水试验(24h)
6. 保护层 → 7. 贴砖
```

## 七、CAD大样图标准

### 7.1 大样编号
```
DT-01  吊顶标准节点
DT-02  跌级吊顶灯槽
FL-01  地砖铺贴
FL-02  木地板铺贴
WL-01  墙面涂料
WL-02  墙面瓷砖
WL-03  木饰面
DR-01  门套剖切
WN-01  窗套节点
FN-01  卫生间防水
```

### 7.2 大样图层
| 图层 | 内容 |
|------|------|
| A-WALL | 墙体剖切面 |
| A-HATCH | 不同材料填充 |
| A-DIMS | 尺寸标注 |
| A-TEXT | 材料标注/说明 |
| A-ANNO | 索引符号 |

### 7.3 大样比例
```
推荐: 1:5 / 1:10（节点大样）
      1:20（局部剖面）
```
