# Guoxin CAD 标准 — 03 标注规范

## 适用范围

本标准适用于所有 Guoxin CAD 图纸的尺寸标注。
从样式设置到标注内容，须统一执行。

---

## 1. 标注样式设置

### 主标注样式：GUOXIN-DIM

```
命令: DIMSTYLE (或 D)
→ 新建 → 样式名: GUOXIN-DIM
→ 用于: 所有标注
```

**线选项卡：**
```
尺寸线颜色: ByLayer (图层: A-DIM → 绿色)
尺寸线线宽: ByLayer (A-DIM → 0.13mm)
基线间距: 8mm
超出标记: 0mm
隐藏尺寸线: 否
```

**符号和箭头选项卡：**
```
第一个箭头: 建筑标记
第二个箭头: 建筑标记
引线箭头: 实心闭合
箭头大小: 2.5mm (打印尺寸)
圆心标记: 无
折断标注: 3.75mm
弧长符号: 标注文字的上方
半径/直径标注的折弯角度: 45°
```

**文字选项卡：**
```
文字样式: GUOXIN-DIM (Romans.shx + gbcbig.shx)
文字颜色: ByLayer (A-DIM)
填充颜色: 无
文字高度: 2.5mm (打印尺寸)
分数高度比例: 1.0
标注文字垂直位置: 上方
标注文字水平位置: 居中
观察方向: 从左到右
从尺寸线偏移: 1mm
```

**调整选项卡：**
```
调整选项: 文字
文字始终保持在尺寸界线之间: 是
在尺寸界线之间绘制尺寸线: 是
全局比例: 按出图比例设置
   1:100 → 100
   1:50  → 50
   1:20  → 20
放置文字的位置: 尺寸线旁边
标注特性比例: 将标注缩放到布局
优化: 手动放置文字
优化: 在尺寸界线之间绘制尺寸线
```

**主单位选项卡：**
```
单位格式: 小数
精度: 0 (整数)
分数格式: 不适用
小数分隔符: 句点(.)
舍入: 0
前缀: 无
后缀: 无
消零: 后续
测量单位比例: 1.0
```

**换算单位选项卡：**
```
不启用 (不常用)
```

**公差选项卡：**
```
方式: 无 (大多数图纸)
必要时: 对称/极限偏差
精度: 0
上偏差: +0.1
下偏差: -0.1
高度比例: 0.6
```

## 2. 标注图层

| 图层 | 颜色 | 线宽 | 内容 |
|------|------|------|------|
| A-DIM | 3 (绿) | 0.13mm | 线性/对齐/半径/角度标注 |
| A-DIMS | 3 (绿) | 0.13mm | 特殊标注（标高、引线等） |

**所有标注线、文字、箭头统一在 A-DIM 图层。**

## 3. 标注规则

### 线性标注

```
         ┌──────────────┐
         │              │
         │              │
  ───────┘              └───────
  │                             │
  │      8000                  │  ← 尺寸数字在上方
  │<──────────────────────────>│
  │                             │
```

- 标注线距图形边界: 8~10mm（打印尺寸）
- 第一道标注线距图形: 10mm
- 连续标注时，相邻尺寸线间距: 8mm
- 尺寸数字在上方居中

### 连续标注顺序

```
  ┌──────┬──────────┬──────────┬──────┐
  │      │          │          │      │
  │ 3000 │   1500   │   2000   │ 1500 │
  │<────>│<────────>│<────────>│<────>│
  │                                     │
  │              8000                   │
  │<───────────────────────────────────>│
```

上排: 分尺寸（逐段标注）
下排: 总尺寸（最外侧）

### 标高标注

```
              ±0.000
            ────●────
                 │
                 ▼
              ┌─────┐
              │     │  ← 完成面标高
              └─────┘
```

- 标高符号: 正三角(等边) + 水平线
- 数字: 精确到小数点后 3 位（±0.000 格式）
- 颜色: A-DIMS 图层，绿色

| 标高类型 | 符号 | 示例 |
|---------|------|------|
| 完成面标高 | 实心三角 | ±0.000 / +2.800 |
| 结构面标高 | 空心三角 | ⚠️ 需标注"结构"二字 |
| 天花底面标高 | 三角+标注 | -0.100（低于完成面） |

### 索引标注

```
            ①
          ┌───┐             ①  大样图编号
          │   │             ─  所在图纸编号（横线=本图）
          └───┘
            │
            └────────────────────────

            ②
          ┌───┐             ②  大样图编号
          │   │             5  所在图纸编号（第5张）
          └─┬─┘
            │
            └────────────────────────
```

### 半径/直径标注

```
  R300 (半径)
  D600 (直径，或 φ600)
```

- 半径用 `R` 前缀，直径用 `⌀` 或 `D` 前缀
- 尺寸线通过圆心（直径）或从圆心引出（半径）

## 4. 特殊标注约定

### 墙体标注

```
          ┌──────┐
   │      │ 200  │      │
   │<─────>│     │<─────>│
   │  墙厚  │     │  墙厚  │
   │       │     │       │
   │<──────┴─────┴───────>│
   │       轴线间距        │
```

### 门洞标注
```
  门洞宽标注在门洞线下方
  门型号标注在门扇旁（如 M-1021）
```

### 窗标注
```
  窗洞宽标注 + 窗台高度标注
  窗型号（如 C-1815，宽1800×高1500）
```

### 房间面积标注
```
  面积标注在房间名称下方
  格式: XX.X m²
  字体: GUOXIN-TEXT，字高3mm
```

## 5. 标注避让规则

当标注重叠或与图形交叉时，按下优先级调整：

```
1. 调整文字位置到尺寸线外侧
2. 适当增加尺寸线间距
3. 断开幕后线（DIMBREAK 命令）
4. 拆分为多段标注
5. 使用引线标注避免遮挡
```

**禁止的做法：**
- ❌ 手动移动标注文字破坏关联性
- ❌ 将标注炸开（Explode）
- ❌ 用 TEXT 手写尺寸数字
