# Guoxin CAD 标准 — 02 文字规范

## 适用范围

本标准适用于所有 Guoxin CAD 图纸的文字标注。
所有文字样式、大小、位置须统一遵守。

---

## 1. 字体约定

| 场景 | 字体 | 宽度因子 | 说明 |
|------|------|---------|------|
| 中文说明 | 黑体（SimHei） | 0.8 | 所有中文标注 |
| 英文/数字 | Romans.shx | 1.0 | 尺寸数字 |
| 中文+英文混合 | 黑体 | 0.8 | 说明文字 |
| 标题 | 黑体（加粗） | 0.8 | 大标题/图名 |

**字体文件要求：**
- 交付他人的 DWG 文件需附带字体文件
- 尽量使用 AutoCAD 自带字体（Romans.shx + 黑体）
- 不使用日文/韩文/生僻字体
- 避免使用 `<Font>` 引用的真字体文件（如 ttf）可能缺失

---

## 2. 文字高度

| 用途 | 1:100 图纸 | 1:50 图纸 | 1:20 细部 | 说明 |
|------|-----------|----------|----------|------|
| 图名（大标题） | 10mm | 7mm | 5mm | 居中加下划线 |
| 图名副标题 | 7mm | 5mm | 3.5mm | 比例/索引 |
| 房间名称 | 5mm | 3.5mm | 2.5mm | 居中放置 |
| 说明文字 | 3.5mm | 3.5mm | 2.5mm | 材料/施工说明 |
| 尺寸数字 | 2.5mm | 2.5mm | 2mm | 使用 DIM 样式 |
| 索引编号 | 3mm | 3mm | 2.5mm | 索引符号内 |
| 标高 | 3mm | 3mm | 2.5mm | 标高符号上方 |
| 图注 | 2.5mm | 2.5mm | 2mm | 图例说明 |
| 标题栏内文字 | 3~5mm | 3~5mm | 3~5mm | 按区域分级 |

> **比例转换公式：** 打印字高 = 图纸字高 × 出图比例
> 例：1:100 出图时，3.5mm 文字在 CAD 中应设为 350mm

---

## 3. 文字样式设置

```
命令: STYLE (或 DDSTYLE)

样式名: GUOXIN-TEXT
  字体: 黑体
  字体样式: 常规
  大字体: 不使用
  高度: 0（使用变量高度）
  宽度因子: 0.8
  倾斜角度: 0
  反向: 否
  倒置: 否
  垂直: 否

样式名: GUOXIN-DIM
  字体: Romans.shx
  大字体: gbcbig.shx
  高度: 0
  宽度因子: 1.0
  （专用于标注样式中的文字）
```

## 4. 文字位置规范

### 房间标注

```
            ┌────────────┐
            │            │
            │   主卧室    │  ← 房间名居中，字高5mm
            │  15.0m²    │  ← 面积标注在下，字高3mm
            │            │
            └────────────┘
```

### 说明文字

```
— 18mm 多层板基层
— 深灰色乳胶漆饰面  ← 说明文字在引线末端
— 成品GRG线条

引线终点与文字起点间距 1mm（打印尺寸）
多行文字行间距：1.5倍字高
```

### 索引标注

```
                       材料/构造说明
                       索引编号
                     ←───┐
                         │
    ─────────────────────┘
          引线指向位置
```

## 5. 文字对齐约定

| 场景 | 对齐方式 | 说明 |
|------|---------|------|
| 图名 | 居中 | 文字居中，下方加下划线 |
| 房间名称 | 居中 | 房间中心位置水平居中 |
| 说明文字 | 左对齐 | 多行接引线 |
| 图注 | 左对齐 | 图例列表 |
| 尺寸数字 | 居中 | 尺寸界线中间（DIM样式控制） |
| 标题栏 | 左对齐/居中 | 按区域模板 |

---

## 6. 禁止事项

- ❌ 文字不放在 0 层
- ❌ 文字不相互重叠
- ❌ 文字不穿入图形
- ❌ 不使用旋转角度（引线标注除外）
- ❌ 不使用炸开文字（Explode text）
- ❌ 文字不超出图框
- ❌ 不使用 CAD 默认的 Standard 文字样式

---

## 7. 常见文字问题检查

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 文字显示 "?" | 缺少字体 | 安装对应字体/修改文字样式 |
| 文字乱码 | 编码不匹配 | 使用统一字体样式 |
| 文字重叠 | 字高太大/间距太小 | 调整字高或断行 |
| 文字在打印时丢失 | 图层设置/颜色打印 | 检查图层打印设置 |
| 中英文高度不一致 | 字体混合的使用 | 统一使用指定样式 |
