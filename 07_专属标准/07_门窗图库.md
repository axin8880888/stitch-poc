## 三、CAD制图符号

### 3.1 门图例
```
单开门:  ─────   ╰──   (90°弧线，半径=门宽)
            门扇

双开门:  ─────   ╰──╰──
                 门扇 门扇

推拉门:  ───┬──  ──┬───  
               门扇  门扇  (夹在墙内)

折叠门:  ───/──/──/──    (锯齿线)
```

### 3.2 窗图例
```
普通窗:  ──══──    (粗线=玻璃，细线=窗框)

飘窗:    ──════──   (三面玻璃)

转角窗:  ╔══╗       (L型)
         ║  ║
```

### 3.3 门弧绘制指南（AutoCAD）
```
单开门弧:
  1. ARC命令
  2. 圆心 = 门轴
  3. 起点 = 门轴，半径 = 门宽
  4. 角度 = 90°
  5. 图层 = A-DOOR

推拉门轨道:
  - 双线平行（轨道宽=门扇厚+20mm）
  - 门扇位置用虚线表示开启
```

## 五、特殊门窗

### 5.1 防火门
| 等级 | 尺寸 | 材质 | 配置 |
|------|------|------|------|
| 乙级 | 900/1000×2100 | 钢质/木质 | 闭门器+顺序器 |
| 甲级 | 1000/1200×2100 | 钢质 | 闭门器+防火锁 |

### 5.2 无障碍门
- 净宽 ≥ **900mm**（轮椅可通过）
- 门把手高度 **900~1000mm**
- 推拉力 ≤ **25N**

### 5.3 百叶门
- 常用于：**燃气表/热水器/管道井**
- 百叶间距：**10~15mm**
- 通风面积 ≥ **30%**
