# 05_线宽规范

> Guoxin CAD Standard — 线宽配置
> 目的：保证打印出图时线型层次分明

---

## 一、线宽分级

### 1.1 五级线宽体系

| 级别 | 线宽(mm) | 打印效果 | 用途 |
|------|---------|---------|------|
| **特粗** | **0.50** | 最醒目 | 被剖切墙体、柱子填充 |
| **粗** | **0.35** | 醒目 | 墙体轮廓、楼梯、图框 |
| **中** | **0.25** | 一般 | 门窗框、家具轮廓、设备 |
| **细** | **0.18** | 辅助 | 标注线、引出线、填充边界 |
| **特细** | **0.13** | 最淡 | 尺寸线、轴线、网格、填充 |
| **极细** | **0.05** | 几乎不可见 | 辅助线（不打印） |

### 1.2 简化版（室内常用）

如果觉得5级太多，可以使用**简化3级**：

| 线宽 | 用途 |
|------|------|
| **0.35** | 墙体、柱子、图框、标题栏 |
| **0.18** | 门窗、家具、设备、标注 |
| **0.13** | 尺寸、填充、引出线 |

> 💡 室内施工图0.35/0.18/0.13三级足够了

## 三、AutoCAD设置方法

### 3.1 图层线宽设置（推荐）
```
LAYER → 选中图层 → Lineweight → 选择对应线宽
```

### 3.2 打印样式表(CTB)设置
```
Page Setup → Plot Style Table → 编辑CTB
→ 按颜色号设置线宽
```

### 3.3 CTB常用配置
```
Color 1-6:  Lineweight = 0.18mm
Color 7:    Lineweight = 0.35mm  (墙体最粗)
Color 8-9:  Lineweight = 0.13mm
Color 10+:  Lineweight = 0.13mm
```

## 四、线宽经验法则

```
墙上图框 = 0.35（最粗，抓眼球）
门窗家具 = 0.18（适中，能看清）
尺寸标注 = 0.13（细，不喧宾夺主）
轴线网格 = 0.05（极细，背景信息）
```

> 打印时：**粗的是结构，中是围护，细是信息**
