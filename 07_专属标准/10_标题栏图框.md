# Guoxin CAD 标准 — 10 标题栏/图框

## 总则

所有 Guoxin CAD 图纸必须使用统一图框和标题栏。
本标准定义图框尺寸、标题栏内容及排版。

---

## 1. 图框尺寸

### 图幅标准

| 图幅 | 外框 (mm) | 内框 (mm) | 装订边 | 其余边距 |
|------|----------|----------|--------|---------|
| A0 | 841×1189 | 821×1169 | 25mm | 10mm |
| A1 | 594×841 | 574×821 | 25mm | 10mm |
| A2 | 420×594 | 400×574 | 25mm | 10mm |
| A3 | 297×420 | 287×410 | 25mm | 5mm |
| A4 | 210×297 | 200×287 | 25mm | 5mm |

## 2. 标题栏

### 标题栏位置

- **固定位置：** 图框右下角
- **A0~A2：** 高度 40~50mm，宽度与内框等宽
- **A3/A4：** 高度 30~40mm，宽度与内框等宽

### 标题栏布局

```
  ┌──────────┬──────────┬──────────┬──────────┬──────┬──────┐
  │          │          │          │          │      │      │
  │  设计单位 │     项目名称     │  比例    │ 图幅 │ 图号 │
  │          │          │          │          │      │      │
  ├──────────┤          ├──────────┼──────────┼──────┼──────┤
  │  制图     │          │  设计     │  审定    │ 日期 │ 版次 │
  │  (签名)   │  图纸名称 ├──────────┼──────────┤      │      │
  │          │          │  校对     │  批准    │      │      │
  ├──────────┤          ├──────────┼──────────┼──────┼──────┤
  │  工程编号 │          │  专业     │  图别    │ 共_页 │ 第_页 │
  └──────────┴──────────┴──────────┴──────────┴──────┴──────┘
```

### 标题栏内容（按区域）

| 区域 | 内容 | 说明 |
|------|------|------|
| 1 | 设计单位 | 公司全称或简称 |
| 2 | 项目名称 | 完整项目名称 |
| 3 | 图纸名称 | 当前图纸名称（如"一层平面图"） |
| 4 | 比例 | 1:100 / 1:50 |
| 5 | 图幅 | A0~A4 |
| 6 | 图号 | 如 JS-01（建施-01） |
| 7 | 制图 | 画图人姓名 |
| 8 | 设计 | 设计师姓名 |
| 9 | 校对 | 校对人员 |
| 10 | 审定 | 审定人 |
| 11 | 批准 | 批准人 |
| 12 | 日期 | 出图日期（YYYY-MM-DD） |
| 13 | 版次 | 初版/V1.0/Rev.A |
| 14 | 工程编号 | 项目内部编号 |
| 15 | 专业 | 建筑/结构/暖通/电气/给排水 |
| 16 | 图别 | 建施/结施/水施/电施 |
| 17 | 页数 | 共 X 页 第 Y 页 |

## 3. 图签（会签栏）

### 会签栏位置

A0~A2 图纸可在图框左上角或标题栏左侧增设会签栏。

## 4. 图纸编号规则

### 编号格式

```
专业码-流水号
```

### 专业码

| 专业 | 代号 | 说明 |
|------|------|------|
| 建筑 | JZ | 建施 |
| 结构 | JG | 结施 |
| 暖通 | NT | 暖施 |
| 给排水 | PS | 水施 |
| 电气 | DQ | 电施 |
| 室内 | SN | 室施（室内设计专用） |

### 编号示例

```
SN-01    一层平面图
SN-02    天花布置图
SN-03    地面铺装图
SN-04    A立面图
SN-05    B立面图
SN-06    C立面图
SN-07    D立面图
SN-08    节点详图（一）
SN-09    节点详图（二）
```

## 5. 其他图框元素

### 图框外标注

- 图纸折叠标记（A4折图线）
- 条形码/二维码（必要时）
- 图纸修改记录表（连续修改时）

### 图纸方向标记

- 指北针（平面图必备）
- 指北针位置：图纸右上角或左上角
- 指北针样式：⊙ 内 N，箭头指向北

### 图例

- 图例位置：图纸左下角或右下角，标题栏上方
- 图例内容：墙体/门/窗/家具等所有图层的标注示例

---

## 6. 图纸命名规范

### 文件名

```
项目简称_图号_图纸名称.dwg

示例:
金沙美容_SN-01_一层平面图.dwg
住宅改造_SN-02_天花布置图.dwg
```

### 文件版本

- 初稿: `_v1.dwg`
- 修改稿: `_v2.dwg`
- 终稿: `_final.dwg`
- 修改后终稿: `_revA.dwg`
