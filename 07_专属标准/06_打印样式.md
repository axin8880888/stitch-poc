# 06_打印样式

> Guoxin CAD Standard — 打印/出图配置
> 适用: AutoCAD DWG To PDF.pc3 → A3/A4 黑白打印

---

## 一、打印样式方案

### 1.1 黑白打印（默认推荐）

```
样式表: monochrome.ctb（AutoCAD自带）
效果:   所有颜色→黑色，线宽区分主次
适用:   施工图/归档图
```

### 1.2 灰度打印（方案/汇报）

```
样式表: grayscale.ctb（AutoCAD自带）
效果:   颜色转灰度，保留层次
适用:   方案册/业主汇报
```

### 1.3 彩色打印（彩色出图）

```
样式表: 自定义 acad.ctb
效果:   保留屏幕颜色
适用:   彩色平面图/彩色立面
```

## 二、标准打印配置（A3 1:50黑白）

### 2.1 Page Setup 参数

| 参数 | 设置 |
|------|------|
| 打印机/绘图仪 | DWG To PDF.pc3 |
| 图纸尺寸 | ISO A3 (420×297mm) |
| 打印区域 | **范围** (Extents) |
| 打印偏移 | **居中打印** |
| 打印比例 | **1:50** 或 **布满图纸** |
| 打印样式表 | **monochrome.ctb** |
| 着色打印 | 按显示 |
| 质量 | **高** |
| 方向 | **横向** |
| 图纸深度 | 按显示 |

### 2.2 CTB 编辑要点（monochrome.ctb）

```
格式:     所有颜色 → 黑色
抖动:     关
灰度:     关
笔号:     自动
虚拟笔号:  自动
淡显:     100

线宽设置:
  Color 1-6, 8-255:  0.18mm
  Color 7:           0.35mm
  启用线宽:          是
```

## 三、比例匹配表

| 图纸比例 | A3纸张尺寸 | 模型空间单位 | 文字字高(CAD) | 打印字高 |
|---------|-----------|-------------|--------------|---------|
| **1:50** | 420×297mm | 21000×14850mm | 3.0mm | 3.0/50=0.06mm |
| **1:100** | 420×297mm | 42000×29700mm | 3.0mm | 3.0/100=0.03mm |
| 1:20 | 420×297mm | 8400×5940mm | 2.5mm | 0.125mm |

> 💡 **实操要点：**
> - 1:50 时CAD文字设 2.5~3.0mm，A3上看正合适
> - 1:100 时CAD文字设 3.0~3.5mm（太小打印看不清）
> - 节点大样 1:20 设 2.5mm

## 四、批量打印流程

### 4.1 用 LISP（推荐）
```
LB-PLOT → 自动遍历所有布局并打印PDF
```

### 4.2 用 SCR 脚本
```
SCRIPT → 批量打印A3.scr
（最多3个布局）
```

### 4.3 手动
```
File → Plot → Page Setup → 选已配置的方案
→ Apply to Layout → Plot
```

## 五、常见问题

### 5.1 打印出来墙体太细
```
原因: Color 7 线宽没设 0.35mm
解决: 编辑CTB，Color 7 线宽=0.35mm
```

### 5.2 轴线打印太显眼
```
原因: Color 2(黄) 在 monochrome 中和墙一样黑
解决: 
  方案1: 在CTB中设 Color 2 淡显=30%
  方案2: 轴线用极细线 0.05mm
  方案3: 自定义CTB单独处理黄色
```

### 5.3 打印有颜色
```
原因: 用了非黑白CTB，或颜色没被映射
解决: 确保用 monochrome.ctb
```

### 5.4 填充打印成纯黑
```
原因: HATCH用SOLID且颜色=7
解决: HATCH用Color 8(灰色)而不是7(黑色)
```

## 七、打印前自检清单

```
□ 所有图元使用 ByLayer 颜色
□ 墙体 Color 7
□ 轴线 Color 2（可在CTB中淡显）
□ 尺寸 Color 3
□ 文字 Color 6
□ 填充 Color 8（不要用7）
□ CTB = monochrome.ctb
□ 比例正确（1:50/1:100）
□ 图纸尺寸正确（A3/A2）
□ 居中打印
□ 预览检查无误
```
