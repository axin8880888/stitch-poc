## 标准制定原则

1. **一致性：** 所有图纸使用同一套标准
2. **可读性：** 任何人拿到你的图纸都能看懂
3. **高效性：** 标准不繁琐，可快速执行
4. **可扩展性：** 随着经验增长持续优化

---

## 图层命名标准（初稿）

完整规范见 `03_规范/001_制图规范.md`

核心原则：
- 前缀 `A-` 表示建筑
- 第二部分表示类型：WALL / DOOR / FURN ...
- 所有图层名使用英文大写
- 不包含多余空格或特殊符号

---

## 制定流程

1. 先使用 `03_规范/` 中的参考规范
2. 在每个项目中验证规范的可用性
3. 发现不合理的地方记录到修改记录
4. 定期（每月）统一更新 Guoxin 标准
5. 标准成熟后固化到模板文件中（DWT）
