## 标准制定原则

1. **一致性：** 所有图纸使用同一套标准
2. **可读性：** 任何人拿到你的图纸都能看懂
3. **高效性：** 标准不繁琐，可快速执行
4. **可扩展性：** 随着经验增长持续优化

---

## 当前已生效标准

| 标准 | 核心内容 |
|------|---------|
| 01_图层标准 | 建筑30+图层定义（颜色/线型/线宽），改造专用图层 |
| 02_文字规范 | 字体约定、字高分级（1:100/1:50/1:20）、对齐规则 |
| 03_标注规范 | GUOXIN-DIM 样式参数、线/箭头/文字/单位完整配置 |
| 10_标题栏图框 | 图幅尺寸、标题栏布局、图纸编号规则、模板计划 |

---

## 制定流程

1. 先使用 `03_规范/` 中的参考规范
2. 在每个项目中验证规范的可用性
3. 发现不合理的地方记录到修改记录
4. 定期（每月）统一更新 Guoxin 标准
5. 标准成熟后固化到模板文件中（DWT）
