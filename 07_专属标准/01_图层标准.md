# Guoxin CAD 标准 — 01 图层命名标准

## 适用范围

本标准适用于 小Claw 参与的所有 CAD 制图项目。
所有图纸中创建的对象必须使用本标准定义的图层。

---

## 通用规则

1. 图层命名格式：`<专业前缀>-<类型>[-<子类>]`
2. 专业前缀统一使用 **大写英文**：`A-` = 建筑，`S-` = 结构，`M-` = 暖通，`P-` = 给排水，`E-` = 电气
3. 类型部分使用 **大写英文**，不超 10 个字符
4. 禁止在 0 层画图（除块定义外）
5. 多余图层使用 `PU` 命令清理

## 图层使用优先级规则

当同一图形元素可能出现在多个图层时，按以下优先级选：

```
墙体 → A-WALL
门窗 → A-DOOR / A-WINDW
固定家具 → A-FURN-FIX
活动家具 → A-FURN
标注 → A-DIM
文字 → A-TEXT
轴线 → A-AXIS
填充 → A-HATCH
```

> ❌ 不要在 A-TEXT 图层画线
> ❌ 不要在 A-DIM 图层写字
> ✅ 每个对象只在一个图层上
