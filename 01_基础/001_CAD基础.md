# 第一阶段：CAD 基础

**学习目标：能够阅读任何一张 CAD 图纸。**

## 1.1 坐标系统

### 世界坐标（WCS - World Coordinate System）
- CAD 的绝对坐标系，固定不变
- X 轴：水平方向
- Y 轴：垂直方向
- Z 轴：垂直于屏幕方向

### 用户坐标（UCS - User Coordinate System）
- 可移动、旋转的自定义坐标系
- 方便在斜面上或不同视角下绘图
- 命令：`UCS`

### 坐标输入方式
- **绝对坐标：** `X,Y`（如 `100,200`）
- **相对坐标：** `@X,Y`（如 `@50,0` 表示向右 50）
- **极坐标：** `@距离<角度`（如 `@100<45`）

## 1.2 单位与比例

### 单位设置
- 命令：`UNITS`
- 室内设计常用：**毫米（mm）**
- 建筑图纸常用：**毫米（mm）**
- 总平面图常用：**米（m）**

### 比例
- **绘图比例：** 通常 1:1 绘制实际尺寸
- **出图比例：** 通过布局空间视口缩放
- **常用出图比例：**
  - 平面图：1:50, 1:100
  - 立面图：1:30, 1:50
  - 节点大样：1:5, 1:10, 1:20
  - 总平面图：1:200, 1:500

| 图幅 | 尺寸 (mm) | 常用场景 |
|------|-----------|---------|
| A0 | 841×1189 | 总平面、大型项目 |

| A4 | 210×297 | 说明、表格、图签 |

## 1.4 模型空间与布局空间

### 模型空间（Model Space）
- **用途：** 1:1 绘制所有图形内容
- **特点：** 无限大的绘图区域
- **切换：** 模型选项卡

### 布局空间（Layout / Paper Space）
- **用途：** 设置图纸布局、图框、标注
- **特点：** 模拟实际纸张大小
- **视口（Viewport）：** 显示模型空间的窗口
- **切换：** 布局选项卡

### 视口比例设置
- 选中视口边框 → 属性面板设置比例
- 命令：`VPSCALE`
- 锁定的视口：防止误缩放

## 2. 图层（Layer）

### 图层作用
- 组织和管理图形对象
- 控制对象可见性
- 控制打印输出
- 统一修改属性

### 图层命名规范（参考）
```
图层类型   命名示例
墙体       A-WALL
家具       A-FURN
门         A-DOOR
窗         A-WINDW
标注       A-DIM
尺寸       A-DIMS
文字       A-TEXT
辅助线     A-DOTE
隐藏线     A-HIDD
中心线     A-CNTR
打印层     A-PLOT
```

### 图层属性
| 属性 | 说明 | 推荐设置 |
|------|------|---------|
| 颜色 | 区分不同元素 | 墙体 7(白)，家具 4(青)，标注 3(绿) |
| 线型 | 区分线型 | 实线 Continuous，虚线 Dashed |
| 线宽 | 区分粗细 | 墙体 0.3-0.5mm，家具 0.13-0.18mm |
| 打印 | 控制是否打印 | 辅助线关，其他开 |

### 图层判断练习
- **墙体：** 双线/单线，颜色突出，图层名含 WALL
- **家具：** 标准块/图例，图层名含 FURN
- **标注：** 含尺寸/文字/箭头，图层名含 DIM
- **文字：** 纯文字说明，图层名含 TEXT

## 3. 常见对象（Object）

| 对象 | 英文 | 说明 |
|------|------|------|
| 直线 | Line | 两点确定一条线 |
| 多段线 | Polyline / PL | 连续线段，整体编辑 |
| 圆 | Circle | 圆心+半径/直径 |
| 圆弧 | Arc | 三点确定弧 |
| 样条曲线 | Spline | 平滑曲线 |
| 椭圆 | Ellipse | 标准椭圆 |
| 图案填充 | Hatch | 剖面线/填充 |
| 块 | Block | 组合对象，整体引用 |
| 属性 | Attribute | 块的附加文字信息 |
| 单行文字 | Text | 单行注释 |
| 多行文字 | Mtext | 多行排版文字 |
| 标注 | Dimension | 尺寸标注 |
| 引线 | Leader | 带箭头的注释引导线 |
| 视口 | Viewport | 布局中的显示窗口 |

### Block 特殊说明
- 块是 CAD 最重要的复用机制
- 修改一个块定义，所有引用同步更新
- 常用块：门窗、家具、图框、标高符号

## 4. 图纸类型

### 建筑平面图
- 全屋布局、墙体、门窗、尺寸标注

### 拆砌墙图
- 需要拆除或新砌的墙体标记

### 天花图（吊顶图）
- 吊顶造型、标高、灯具、空调口

### 地面铺装图
- 地砖/地板铺装方向、尺寸、材料

### 立面图
- 墙面造型、材质、尺寸（正交投影）

### 节点图
- 构造细部放大（如收口、转角）

### 剖面图
- 从某个位置切开后的内部构造

### 大样图
- 某个局部的详细放大图

### 家具图
- 定制家具的三视图/结构图

### 门头图
- 店面门头造型、招牌尺寸、材质

### 施工图
- 综合以上所有：指导实际施工的完整图纸

2. 在图纸中找出上述所有对象类型
3. 练习切换模型/布局空间，查看视口设置
