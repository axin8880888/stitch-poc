# 门头设计规范

## 总则

门头（店铺门面）是商铺设计的核心。好的门头既符合城管法规，又能吸引客流。
本文汇总门头设计中常用的尺寸标准、招牌规范、结构要求和审批要点。

---

## 1. 门头尺寸标准

### 常用门头尺寸

| 门头类型 | 宽度范围 | 高度范围 | 适用场景 |
|---------|---------|---------|---------|
| 窄门头 | 2000~4000 | 1000~1500 | 沿街小铺、档口 |
| 标准门头 | 4000~8000 | 1500~2500 | 沿街商铺、美容院 |
| 大门头 | 8000~15000 | 2500~5000 | 商场入口、品牌旗舰店 |
| 转角门头 | 视建筑 | 1000~3000 | 临街转角位（两个面） |
| 内嵌门头 | 视门洞 | 800~1500 | 商场内铺、走廊商铺 |

### 门头高度与建筑关系

```
    建筑外立面
  ┌─────────────────────────┐
  │                         │
  │   上层窗户/幕墙          │
  │                         │
  ├─────────────────────────┤ ← 门头上沿（一般与二层楼板齐平）
  │   ┌─────────────────┐   │
  │   │   门头招牌        │   │ ← 门头高度 H
  │   │   ××美容         │   │
  │   └─────────────────┘   │
  ├─────────────────────────┤ ← 门头下沿（一般做铝塑板包边）
  │   门  │  橱窗  │  门    │ ← 入口区
  │       │        │        │
  └───────┴────────┴────────┘
          地面

H 推荐：1200~2000mm（根据层高调整）
门头下沿距地：2500~3000mm（保证净高）
```

### 门头与建筑的关系

| 项目 | 规范 | 说明 |
|------|------|------|
| 门头上沿 | 不超二层窗台 | 不得遮挡上层窗户 |
| 门头挑出 | 一般≤300mm | 超出建筑红线需审批 |
| 侧招 | 与墙面夹角≤30° | 垂直于墙面 |
| 门头覆盖 | ≤该面墙面积的60% | 部分地区有具体规定 |

## 2. 招牌规范

### 招牌类型

| 类型 | 特点 | 造价 | 适用 |
|------|------|------|------|
| 发光字 | 夜间醒目，LED/霓虹 | 💰💰 | 所有商铺 |
| 亚克力字 | 质感好，可打光 | 💰💰 | 美容院、高端 |
| 金属字（不锈钢/铜）| 质感高级，耐用 | 💰💰💰 | 品牌店、牙科 |
| 喷绘布 | 便宜，可换 | 💰 | 临时活动、工地围挡 |
| UV打印 | 色彩丰富，精度高 | 💰 | 奶茶店、快速消费 |
| 灯箱（吸塑/拉布）| 整面发光 | 💰💰 | 药店、便利店 |
| LED屏 | 动态内容 | 💰💰💰💰 | 商场入口 |

### 字体尺寸

| 招牌高度 | 最大字高 | 推荐字高 | 说明 |
|---------|---------|---------|------|
| 600mm | 400mm | 300~350mm | 小门头 |
| 800mm | 550mm | 400~500mm | 标准门头 |
| 1000mm | 700mm | 500~600mm | 中大门头 |
| 1200mm | 850mm | 600~750mm | 大门头 |
| 1500mm | 1000mm | 700~900mm | 超大 |

**字间距建议：** 字宽的 1/4 ~ 1/3

### 招牌颜色搭配

| 商业类型 | 推荐配色 | 说明 |
|---------|---------|------|
| 美容/SPA | 白+金/粉+白 | 柔和高级感 |
| 餐饮 | 红+黄/橙+黑 | 激发食欲 |
| 奶茶/甜品 | 粉+白/蓝+白 | 年轻化 |
| 品牌服饰 | 黑白/品牌色 | 简约有格调 |
| 药店 | 绿+白 | 行业通用 |
| 牙科 | 蓝+白 | 干净专业 |
| 教育机构 | 蓝/绿+白+橙 | 信任+活力 |

### LOGO 位置
- LOGO 一般放在招牌 **最左或居中**
- LOGO 高度一般为招牌高度的 60%~80%
- LOGO 两侧留白 ≥ 一个LOGO宽度（推荐）

## 3. 门头结构要求

### 骨架结构

| 类型 | 材料 | 说明 |
|------|------|------|
| 轻钢龙骨 | 镀锌方管 40×20/40×40 | 轻型招牌 |
| 角钢骨架 | L30~L50 角钢 | 中型招牌 |
| 方管钢架 | 40×40/50×50 方管 | 重型招牌（悬挑） |
| 膨胀螺栓 | M10/M12 | 固定在墙体上 |
| 化学锚栓 | M12/M16 | 空心砖/轻质墙专用 |

### 结构安全要点

1. **所有招牌必须可靠固定在结构墙体上**
2. 空心砖墙必须使用 **化学锚栓** 或穿墙螺栓
3. 悬挑招牌（>300mm）需做 **钢结构计算**
4. 长距离招牌（>6m）需设 **伸缩缝**
5. **防雷接地**：金属招牌需接建筑防雷系统
6. 招牌支架须做 **防锈处理**（热镀锌/防锈漆）

### 荷载参考

| 招牌类型 | 自重 | 风荷载（按地区） |
|---------|------|----------------|
| 亚克力字+底板 | 15~30 kg/m² | 0.3~0.8 kN/m² |
| 灯箱 | 20~40 kg/m² | 同上 |
| 金属字+铝塑板 | 20~50 kg/m² | 同上 |
| 大型钢架+饰面 | 50~100 kg/m² | 同上 |

> 具体风荷载需按 GB 50009-2012《建筑结构荷载规范》计算

## 4. 招牌灯箱规格

### 吸塑灯箱

| 项目 | 规格 | 说明 |
|------|------|------|
| 灯箱厚度 | 80~180mm | 常见120~150mm |
| 光源 | LED 模组 | 防水等级 IP65 |
| 灯珠间距 | 80~120mm | 均匀发光 |
| 面板 | 亚克力板 2~3mm | 可吸塑成型 |
| 底板 | 铝塑板/镀锌板 | 防水处理 |

### 发光字（背面发光/正面发光）

| 项目 | 规格 | 说明 |
|------|------|------|
| 字壳 | 不锈钢/铝型材 | 1.0~1.5mm厚 |
| 光源 | LED 模组 | 防水，12V/24V |
| 亚克力面板 | 3~5mm | 乳白亚克力匀光 |
| 电源 | 防水开关电源 | 放在招牌内部 |
| 安装方式 | 明装/暗装 | 暗装更美观 |

## 5. 门头审批要点

### 城管审批基本要求

| 项目 | 要求 | 说明 |
|------|------|------|
| 审批主体 | 当地城市管理局 | 部分区划需街道办初核 |
| 申请材料 | 立面效果图+结构图+位置图 | 需盖设计章（部分地区） |
| 审批时限 | 5~15个工作日 | |
| 有效期 | 1~2年（续期） | 到期需重新申请 |
| 审批费用 | 免费（或少量工本费） | 部分地区收押金 |

### 常见违规红线

| ❌ 违规项 | 后果 | 说明 |
|----------|------|------|
| 遮挡二楼窗户 | 强拆+罚款 | 最常见违规 |
| 超出建筑外立面 | 强拆+罚款 | 侵占公共空间 |
| 未审批先行安装 | 强拆+罚款 | 损失全部成本 |
| 遮挡交通标识 | 即拆不赔 | 安全第一 |
| 遮挡消防通道/消防设施 | 即拆+重罚 | 严重安全隐患 |
| 灯光直射住宅窗户 | 整改+罚款 | 光污染 |
| 面积超标 | 整改或拆除 | 各地标准不一 |

### 审批流程

```
1. 咨询城管局 → 确认当地门头审批要求
2. 准备材料 → 效果图(正面/侧面/鸟瞰)
                结构图(承重/固定方式)
                位置图(在建筑物上的位置)
                营业执照复印件
                产权证明/租赁合同
3. 提交申请 → 窗口/线上
4. 现场勘查 → 城管派人核实
5. 领取许可 → 张贴在店内备查
6. 施工安装 → 按图施工
7. 验收备案 → 部分区域需竣工验收
```

### 各地特殊要求（以当地为准）

- **上海：** 内环内禁止 LED 滚动屏
- **北京：** 长安街沿线招牌高度≤1500mm
- **深圳：** 采用"一店一招"原则
- **广州：** 招牌面积与被覆盖墙体面积比有上限
- **成都：** 商业集中区统一风格

> ⚠️ 每个城市甚至每个街道的审批标准可能不同，**开工前务必咨询当地城管**

## 6. 门头设计实战建议

### 美容院门头（参考第1周项目）

| 项目 | 推荐 | 说明 |
|------|------|------|
| 风格 | 简约现代/轻奢 | 白+金色系 |
| 招牌形式 | 亚克力发光字+底板 | 质感好，造价适中 |
| 门头材料 | 铝塑板/石材 | 耐用，好打理 |
| 灯光 | 3500K~4000K | 暖白光，显皮肤好 |
| 橱窗 | 大幅海报/展示 | 展示项目/作品 |
| 入口 | 感应门/推拉门 | 方便进出 |

### 餐饮门头（后续项目参考）

| 项目 | 推荐 | 说明 |
|------|------|------|
| 风格 | 亮色/品牌色 | 吸引眼球 |
| 招牌形式 | 吸塑灯箱/发光字 | 夜间醒目 |
| 门头材料 | 铝塑板/仿砖 | 耐油烟 |
| 灯光 | 3000K~3500K | 暖光，显食物好 |
| 橱窗 | 半开敞/明档 | 展示制作过程 |
| 入口 | 推拉门/折叠门 | 客流高峰方便 |
