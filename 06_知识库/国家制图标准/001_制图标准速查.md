# 国家制图标准速查（室内装饰设计专用）

> 依据: GB/T 50001-2010《房屋建筑制图统一标准》
>        GB/T 50104-2010《建筑制图标准》
>        GB/T 50103-2010《总图制图标准》
> 提取室内装饰设计最常用的规定

---

## 一、图幅与图框

### 1.1 标准图幅尺寸

| 图幅代号 | 尺寸 (B×L mm) | 图框距边 (装订侧) | 图框距边 (其他三侧) |
|---------|--------------|-----------------|------------------|
| A0 | 841 × 1189 | 25mm | 10mm |
| A1 | 594 × 841 | 25mm | 10mm |
| A2 | 420 × 594 | 25mm | 10mm |
| **A3** | **297 × 420** | **25mm** | **5mm** |
| A4 | 210 × 297 | 25mm | 5mm |

> 💡 **室内设计最常用：A3**（方案册/施工图册标准）

### 1.2 标题栏（图签）
- 最小宽度 **40mm**（A3/A4）/ **50mm**（A0-A2）
- 必须包含：**工程名称、图名、图号、比例、设计/制图/审核人、日期**
- 推荐自下而上排序：设计 → 制图 → 审核 → 审定

## 二、比例

### 2.1 常用比例
| 图纸类型 | 推荐比例 |
|---------|---------|
| 总平面图 | 1:500, 1:1000 |
| 平面图 | **1:50**, 1:100 |
| 立面图 | **1:50**, 1:100 |
| 剖面图 | **1:50**, 1:100 |
| 节点大样 | **1:1**, 1:2, 1:5, 1:10, **1:20** |
| 详图索引 | 1:20, 1:30 |

> 💡 **室内施工图标准：平面/立面/剖面 1:50，大样 1:10~1:20**

### 2.2 比例标注
- 写在图名的右侧（横线下方）
- 格式：`1:50` 或 `1:100`
- 同一张图使用同一种比例时，可统一注在标题栏

## 三、线型与线宽

### 3.1 线宽组（室内装饰推荐）

| 线宽名称 | 宽度 | 用途 |
|---------|------|------|
| **粗线** | **0.5mm** | 被剖切的墙体轮廓、柱子断面 |
| **中粗线** | **0.35mm** | 可见轮廓线、门窗框 |
| **中细线** | **0.25mm** | 家具轮廓、标注线、尺寸界线 |
| **细线** | **0.13mm** | 填充线、引出线、尺寸线、轴线 |
| **特细线** | **0.05mm** | 辅助线、网格线 |

> 💡 **一套图只用1~2种线宽组即可：** 0.35(墙) / 0.18(家具) / 0.13(标注)

### 3.2 线型
| 线型 | 用途 |
|------|------|
| **实线** CONTINUOUS | 可见轮廓线、墙体、家具 |
| **虚线** HIDDEN / DASHED | 不可见轮廓（吊顶以上梁、柜顶） |
| **点划线** CENTER / CENTER2 | 轴线、对称线、中心线 |
| **双点划线** PHANTOM | 原有建筑轮廓、拆除线 |
| **折断线** | 断开界线 |

## 四、文字标注

### 4.1 字高
| 用途 | 字高 (mm) |
|------|-----------|
| 图名（大标题） | **5~7** |
| 图名（小标题） | **4~5** |
| 房间名称 | **3.5** |
| 尺寸数字 | **2.5** |
| 说明文字 | **3.0** |
| 索引编号 | **3.0** |

> 💡 **1:50图纸打印到A3：** 标注字高 = 2.5 × 50 = 125mm（图纸上）
> 所以CAD中设 **字高=2.5** 模型空间，打印时1:50正好看清

### 4.2 字体
- **推荐：** 宋体 / 仿宋（中文），Romans（英文/数字）
- 施工图正文：**仿宋_GB2312**
- 尺寸/标注：**Romans + 宋体**（SHX字体兼容性好）
- ✘ 避免使用：楷体、隶书（笔划太细打印看不清）

## 五、尺寸标注

### 5.1 尺寸组成
```
【尺寸界线 → 尺寸线 → 尺寸起止符 → 尺寸数字】

          │         │         │
   尺寸界线  尺寸线  起止符(45°斜线)
```

### 5.2 标注规则
- **三道尺寸**（从内到外）：
  1. **第一道**：门窗洞口的定位尺寸
  2. **第二道**：轴线间距（墙中到墙中）
  3. **第三道**：建筑外轮廓总尺寸
- 尺寸单位：**mm**（图中不用写单位）
- 尺寸数字离尺寸线 **≥1mm**
- 尺寸界线伸出尺寸线 **2~3mm**
- 起止符号用**45°中粗斜线**，长 **2~3mm**

### 5.3 半径/直径/角度
- 半径：R + 数字（如 R200）
- 直径：**ø** + 数字（如 ø150）
- 角度：**°** + 数字（如 45°）
- 标高符号：**▽** 或 **▲**（三角形+横线）

## 六、符号

### 6.1 索引符号
```
     ┌──────┐
     │  5   │     ← 详图编号
     │──────│
     │  2   │     ← 详图所在图纸号
     └──────┘
```
- 圆直径 **10mm**
- 详图在本页：下半圆用横线 `──`

### 6.2 剖切符号
```
    ──►────       ← 剖切方向（箭头表示投影方向）
    │
    A            ← 剖切编号
```
- 剖切位置线长 **6~10mm**
- 投影方向用**箭头**

### 6.3 引出线
```
  文字说明 ────────●     ← 小圆点（指向标注对象）
              │
             ─┴─    ← 横线
```
- 第一层引出线用**细实线**
- 多层标注可加横线层叠

## 七、标高

### 7.1 表示方法
```
       ±0.000        ← 完成面标高
        ─┴─
```
- ±0.000 = 室内首层完成面
- 正数不写「+」，负数写「−」
- 标高单位：**m**（唯一用米的标注，保留3位小数）

### 7.2 各类标高
| 类型 | 符号 | 举例 |
|------|------|------|
| 完成面标高 | 实心三角形 | ±0.000 |
| 结构标高 | 空心三角形 | -0.050 |
| 室外标高 | 室外符号 | -0.300 |

## 八、图层命名规范（补充）

### 8.1 推荐命名体系
```
A-       建筑类    A-WALL(墙体)  A-DOOR(门)  A-DIMS(尺寸)
P-       设施类    P-FURN(家具)  P-PLMB(给排水)
S-       结构类    S-COLS(柱)    S-DEMO(拆除)
E-       电气类    E-LIGHT(照明) E-POWR(插座)
M-       暖通类    M-DUCT(风管)  M-AC(空调)
```

### 8.2 颜色规范建议
| 颜色 | 用途 |
|------|------|
| 1 (红) | 柱子、拆除线（S-DEMO） |
| 2 (黄) | 轴线（A-AXIS） |
| 3 (绿) | 墙体填充、尺寸（A-DIMS） |
| 4 (青) | 门、窗（A-DOOR, A-WINDW） |
| 5 (蓝) | 给排水（P-PLMB） |
| 6 (紫) | 家具、文字（P-FURN, A-TEXT） |
| 7 (白/黑) | 墙体线（A-WALL）、标题 |
| 8 (灰) | 填充（A-HATCH） |

## 九、图样画法

### 9.1 平面图
- 剖切高度：**窗台上方**（约1.2~1.5m高度水平剖切）
- 内容：墙、柱、门窗、家具、洁具、尺寸、标高、房间名
- 指北针（底层平面）：圆直径 **24mm**，箭头北向

### 9.2 立面图
- 投影方向：**正投影**（不打透视）
- 标注：地面标高、顶面标高、主要材料、尺寸
- 隐藏线：不可见轮廓用**虚线**

### 9.3 剖面图
- 剖切位置选在：楼梯间、门窗洞、结构变化处
- 标注：各层标高、层高、吊顶高度

## 十、常见错误与检查清单

```
□ 图幅是否正确（A3/A2/A1）
□ 图框边距是否标准
□ 标题栏信息是否完整
□ 比例是否正确（1:50/1:100）
□ 三道尺寸是否完整
□ 标高符号/单位是否正确
□ 字体是否统一（仿宋/宋体）
□ 图层颜色是否合规
□ 线宽组是否一致
□ 索引/剖切是否一一对应
□ 房间名是否标注完整
□ 总尺寸=各分段之和
```
