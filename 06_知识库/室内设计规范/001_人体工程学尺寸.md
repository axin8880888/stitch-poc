# 人体工程学关键尺寸

## 总则

人体工程学尺寸是室内设计的底层数据。所有平面布局、立面高度、家具尺寸，都建立在人的活动尺度之上。

**单位约定：** 本文所有尺寸均为 mm，除非特别说明。

## 3. 门洞尺寸

### 标准门洞

| 类型 | 门洞宽 | 门洞高 | 适用场景 |
|------|--------|--------|---------|
| 单开门（窄） | 700 | 2100 | 卫生间、储物间 |
| 单开门（标准） | 800 | 2100 | 卧室、书房 |
| 单开门（宽） | 900 | 2100 | 主卧、大门 |
| 双开门 | 1200~1500 | 2100~2400 | 主入口、大厅 |
| 推拉门（单扇） | 800~1200 | 2100~2400 | 厨房、阳台 |
| 推拉门（双扇） | 1500~2400 | 2100~2400 | 客厅通阳台 |
| 折叠门 | 1200~3000 | 2100~2400 | 大开间隔断 |
| 玻璃地弹簧门 | 900~1200 | 2400~2800 | 商铺门头 |

### 门洞与门扇关系

```
门洞宽 = 门扇宽 + 门框(50~60) × 2     ≈ 门扇宽 + 100~120
门洞高 = 门扇高 + 门框上 + 地面留缝   ≈ 门扇高 + 80~100

示例：800门洞 → 门扇约680~700宽
      900门洞 → 门扇约780~800宽
```

**门洞最小宽度参考：**
- 房门：800（门扇680+）
- 卫生间门：700（门扇580+）
- 厨房门（非开放）：700~800
- 无障碍门：900

### 门高注意事项
- 标准门高 2100（含门框）
- 如有中央空调吊顶，注意门高与吊顶的关系
- 若有梁下翻，注意门洞顶部高度

## 6. 操作台/工作台高度

```
┌─────────────────────────────────────┐
│             舒适高度公式               │
├─────────────────────────────────────┤
│ 操作台 = 身高 × 0.4 - 50            │
│ 示例：身高170cm → 操作台 ≈ 630mm    │
│ 标准用850mm（兼容大多数）              │
│                                     │
│ 写字台 = 身高 × 0.25 + 350          │
│ 示例：身高170cm → 书桌 ≈ 775mm      │
│ 标准用750mm                          │
│                                     │
│ 吧台高度 = 950~1100（配吧台凳）       │
│ 吧台凳面距地 = 750~800               │
└─────────────────────────────────────┘
```

### 各类台面高度速查

| 类型 | 高度 | 备注 |
|------|------|------|
| 书桌/写字台 | 720~760 | 标准750 |
| 电脑桌（键盘位） | 650~700 | 比书桌低 |
| 餐桌 | 720~760 | 标准750 |
| 厨房操作台 | 800~900 | 标准850 |
| 中岛（厨房） | 900~1000 | 配吧台凳 |

## 8. 常用房间最小尺寸

### 客厅
```
开间（墙中）  ≥3600   舒适4000~4800
进深（墙中）  ≥4500   舒适5000~6000
（含沙发区+电视墙+通道）
```

### 主卧
```
开间 ≥3300   舒适3600~4200
进深 ≥4000   舒适4500~5400
（含1.8m床+双床头柜+衣柜+通道）
```

### 次卧
```
开间 ≥3000   舒适3300~3600
进深 ≥3600   舒适4000~4800
（含1.5m床+单床头柜+衣柜+书桌）
```

### 厨房（封闭式）
```
最小面积：5㎡
一字型：宽≥1800   长≥3000
L型：   宽≥1800   长≥2500
U型：   宽≥2400   长≥2500
```

### 卫生间
```
最小面积：3~4㎡
宽度：≥1600（马桶+洗手台）
深度：≥2000（含淋浴/浴缸）
舒适尺寸：2000×2500
干湿分离：宽度需≥2000
```

## 10. 速查表：常见设计错误

| ❌ 常见错误 | ✅ 正确做法 | 后果 |
|------------|-----------|------|
| 床两侧通道<400 | 床侧≥600 | 无法整理床铺 |
| 沙发前距茶几<300 | 400~500 | 腿伸不开 |
| 餐桌距墙<600 | ≥900 | 椅子拉出后人走不过 |
| 衣柜前通道<600 | ≥900 | 开门后站不下人 |
| 马桶中心距墙<350 | 400~450 | 手肘碰墙 |
| 洗手台高度800配身高180| 按身高调整 | 弯腰洗碗腰疼 |
| 电视挂高1600 | 中心线1200 | 仰头看电视难受 |
| 门洞开在墙角<100留边 | ≥100 | 门套装不进去 |
