# 第六阶段：知识库

**目标：持续积累 CAD 制图相关的行业知识、规范和经验。**

---

## 知识分类

### CAD 命令
- 命令手册（详见 02_命令/）
- 常用 LISP 程序
- 常用 SCR 脚本

### 施工规范
- 室内装饰施工规范
- 木工/泥水/水电/油漆工艺

### 门头规范
- 门头尺寸标准
- 招牌规范
- 门头结构要求
- 城管审批要点

### 室内设计规范
- 人体工程学尺寸
- 走道最小宽度
- 门洞尺寸标准
- 楼梯踏步规范
- 厨房/卫生间尺寸

### 材料规范
- 常用材料规格表
- 饰面材料
- 基层材料
- 五金配件

### 消防规范
- 疏散距离
- 防火门要求
- 消防通道宽度
- 喷淋/烟感间距

### 人体工程学（关键尺寸）
```
走道宽度：≥900mm（主通道）
门洞宽度：800/900mm（房门），1500/1800mm（双开门）
桌面高度：750mm
椅子高度：450mm
床尺寸：1500×2000 / 1800×2000
沙发座深：500-600mm
厨房操作台高度：850mm
洗手台高度：800-850mm
```

### 国家制图规范
- GB/T 50001-2010 房屋建筑制图统一标准
- GB/T 50104-2010 建筑制图标准
- GB 50016-2014 建筑设计防火规范

## 新增知识流程

遇到新知识点时：

1. 记录来源（项目经验/查资料/别人教的）
2. 判断分类
3. 写入对应分类文档
4. 在知识库 README 中添加索引
5. 如果重要，更新到专属标准（07_专属标准/）
