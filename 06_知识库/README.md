## 新增知识流程

遇到新知识点时：

1. 记录来源（项目经验/查资料/别人教的）
2. 判断分类
3. 写入对应分类文档
4. 在知识库 README 中添加索引
5. 如果重要，更新到专属标准（07_专属标准/）
