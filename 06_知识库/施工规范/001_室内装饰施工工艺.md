# 室内装饰施工规范（基础）

## 总则

掌握基本的施工工艺和规范，才能画出真正可落地的施工图。本文记录装修施工中最常见的工艺要点和图纸表达规范。

---

## 1. 拆砌墙工程

### 拆除工程

| 项目 | 规范要求 | 备注 |
|------|---------|------|
| 承重墙 | ❌ 严禁拆除 | 图纸中用 A-DEMO 标记拆除时需区分 |
| 非承重墙 | 可拆 | 一般120/100mm隔墙 |
| 梁 | ❌ 严禁拆除 | 即使看起来是后砌的 |
| 剪力墙 | ❌ 严禁拆除 | 混凝土结构墙 |
| 构造柱 | ⚠️ 需结构确认 | 一般在墙体转角/交接处 |
| 拆除方式 | 切割机切缝→电锤打碎 | 勿用大锤砸，防震裂相邻结构 |

**图纸标注规范：**
- A-DEMO（红虚线）：标记拆除范围
- 标注"拆除至结构层"或"拆除至原顶"
- 拆除边界需标注完成面线

### 砌墙工程

| 项目 | 规范 | 说明 |
|------|------|------|
| 墙体类型 | 轻质砖（加气块） | 非承重隔墙 |
| 墙体厚度 | 100/120mm | 常用隔墙 |
| 卫生间墙体 | 120mm实心砖或钢筋混凝土 | 防水要求 |
| 砌筑高度 | 砌至梁底/板底 | 留斜砌（45°~60°） |
| 新老墙交接 | 植入钢筋 | φ6@500~600，L型连接 |
| 挂钢丝网 | 新老墙交界各搭接150mm | 防开裂 |
| 过梁 | 门洞上方必需 | ≥100mm搁置长度 |
| 构造柱 | 墙长＞4m需设 | 抗震要求 |

**图纸标注规范：**
- A-NEWW（青实线）：新建墙体位置
- 标注墙体材质（加气块/实心砖）
- 标注门洞尺寸及过梁
- 标注植筋、挂网要求

### 门洞过梁

```
                  ┌─────────────────┐
                  │    预制/现浇     │
   墙体           │    过梁        │    墙体
                  │  ≥100    ≥100  │
                  └─────────────────┘
                                   
                       门洞
                        
过梁高度：100~120mm
过梁长度 = 门洞宽 + 200mm（两端各搁置100mm）
门洞靠墙边时，单侧搁置≥100即可
```

## 2. 水电工程

### 强电

| 项目 | 规范 | 说明 |
|------|------|------|
| 照明回路 | 1.5mm²铜线 | BV线 |
| 插座回路 | 2.5mm²铜线 | BV线 |
| 厨房/卫生间 | 4mm²铜线 | 单独回路 |
| 空调（1.5匹） | 2.5mm² | 单独回路 |
| 空调（3匹+） | 4mm² | 单独回路 |
| 冰箱 | 2.5mm² | 建议单独回路（出远门不断电） |
| 总进线 | 10mm²/16mm² | 视总功率 |
| 插座高度（普通） | 距地 300mm | 完成面 |
| 插座（电视墙） | 距地 350mm | 隐藏于电视柜后 |
| 开关高度 | 距地 1300~1400mm | 完成面 |
| 空调插座 | 距地 2200~2300mm | 靠近室内机 |
| 厨房台面插座 | 距台面 100~150mm | 多个并排 |

### 弱电

| 项目 | 规范 | 说明 |
|------|------|------|
| 网线 | 超五类/六类 | 建议六类起 |
| 弱电箱 | 距地 300~500mm | 方便检修 |
| 电视线 | 同轴电缆 | 现在多用网线代替 |
| 音箱线 | 预留管 | 穿管走线 |

### 给排水

| 项目 | 规范 | 说明 |
|------|------|------|
| 给水管 | PPR管 De20/De25 | 热水管包保温 |
| 排水管 | PVC De50/De110 | |
| 冷热水 | 左热右冷 | 面向墙面判断 |
| 热水管 | 包保温棉 | 防冷凝/保温 |
| 排水坡度 | ≥1%（每米降10mm） | 排水管 |
| 卫生间排水 | 墙排/地排 | 墙排美观，地排防堵 |
| 存水弯 | 必需 | 防臭 |
| 地漏 | 一个排水点一个地漏 | 淋浴区必备 |

## 3. 防水工程

### 防水区域

| 区域 | 防水高度 | 做法 | 说明 |
|------|---------|------|------|
| 卫生间地面 | 满做 | 2~3遍涂刷 | 上翻墙面至少300mm |
| 卫生间淋浴区 | 满做到1800mm | 2~3遍涂刷 | 墙上防水 |
| 卫生间洗手台 | 墙面防水到1200mm | | |
| 厨房地面 | 满做 | 2遍涂刷 | 上翻300mm |
| 阳台地面 | 满做 | 2遍涂刷 | 上翻300mm |
| 地下室 | 满做 | 特殊防水 | 需专业队伍 |

### 闭水试验
- 防水层干透后（通常24h）
- 蓄水深度：20~30mm
- 闭水时间：48h
- 前往楼下观察无渗漏
- 合格后方可铺贴

| 轻钢龙骨间距 | 300~400mm | 主龙骨 |

## 7. 竣工图纸记录要点

**施工完成后，工程图应标注以下内容：**

1. **给水点位** — 热水器、洗手台、马桶、洗衣机、厨房水槽
2. **排水点位** — 地漏位置、排水立管、存水弯
3. **强电点位** — 插座、开关、空调、灯具
4. **弱电位** — 网口、电视、电话、安防
5. **墙体改造** — 拆除/新建与实际一致
6. **吊顶范围** — 标高、材质、检修口
7. **防水范围** — 防水区域和高度

> **竣工图 = 施工过程记录 + 现场实际修改 → 保留给业主和物业**
