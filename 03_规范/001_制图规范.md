## 2. 文字规范

| 用途 | 图纸比例 | 字高(mm) |

| 中标题 | 1:100 | 5 |

| 大标题 | 1:50 | 5 |

### 2.3 文字样式设置
```
命令: STYLE → 新建 → 样式名 "GUOXIN-TEXT"
字体名: 黑体
字体样式: 常规
高度: 0 (使用变量高度)
宽度因子: 0.8
倾斜角度: 0
```

## 3. 标注规范

### 3.1 标注样式
```
命令: D → 新建 → 样式名 "GUOXIN-DIM"
```

**线：**
- 基线间距：8mm
- 超出尺寸线：1.5mm
- 起点偏移量：2mm

**符号和箭头：**
- 第一个箭头：建筑标记
- 第二个箭头：建筑标记
- 引线：实心闭合
- 箭头大小：2.5mm

**文字：**
- 文字样式：Romans（数字）+ 黑体（中文前缀）
- 文字高度：2.5mm
- 文字位置：垂直上方 / 水平居中

**调整：**
- 文字始终保持在尺寸界线之间
- 标注特征比例：使用全局比例（按出图比例）

**主单位：**
- 单位格式：小数
- 精度：0
- 小数分隔符：句点
- 舍入：0
- 前缀/后缀：无

### 3.2 标注图层规范
- 标注放在 A-DIM 图层
- 标注线颜色：3（绿色）
- 标注线宽：0.13mm

## 4. 出图规范

### 4.1 图框与标题栏

**图框尺寸：**
```
A0: 841×1189mm  内框: 821×1169mm  装订边: 25mm  其余: 10mm
A1: 594×841mm   内框: 574×821mm   装订边: 25mm  其余: 10mm
A2: 420×594mm   内框: 400×574mm   装订边: 25mm  其余: 10mm
A3: 297×420mm   内框: 287×410mm   装订边: 25mm  其余: 5mm
A4: 210×297mm   内框: 200×287mm   装订边: 25mm  其余: 5mm
```

**标题栏内容（从左到右）：**
1. 项目名称
2. 图纸名称
3. 图号
4. 设计单位
5. 制图 / 审核 / 批准
6. 比例
7. 日期
8. 图幅规格

### 4.2 输出格式

**PDF：**
- 使用 DWG To PDF 打印机（推荐）
- 或 PDFCreator / Adobe PDF
- 合并图层为 Off（打印为光栅）

**DWG：**
- 保存版本：AutoCAD 2013 / 2018（通用性最高）
- 使用命令：`SAVEAS`

**DXF：**
- 用于其他软件对接
- 保存版本：AutoCAD 2013 DXF
- 使用命令：`DXFOUT`

### 4.3 打印样式表（CTB/STB）
- 推荐使用 **命名打印样式（STB）**
- 按图层指定颜色、线宽
- 颜色设为"对象颜色"（使用图层颜色）
- 线宽设为"使用对象线宽"

## 5. 常见规范检查清单

### 图层检查
- [ ] 所有对象在正确图层上
- [ ] 没有对象在 0 层（除块定义外）
- [ ] 图层命名统一规范
- [ ] 多余图层已清理（PU）

### 文字检查
- [ ] 使用统一字体
- [ ] 文字高度符合图纸比例
- [ ] 文字不重叠、不遮挡图形
- [ ] 无乱码（中文字体正确安装）

### 标注检查
- [ ] 标注样式统一
- [ ] 标注尺寸正确（与实际一致）
- [ ] 标注不重叠
- [ ] 箭头大小统一

### 出图检查
- [ ] 图框正确
- [ ] 标题栏信息完整
- [ ] 比例正确
- [ ] 页面设置正确
- [ ] 打印样式已配置
- [ ] 所有布局已设置
- [ ] 图纸已经 PURGE 清理

---

**学习建议：**
- 每画一张图后，对照本检查清单逐项核对
- 逐步形成检查习惯，不需要背清单
- 当发现不符合规范的地方，记录下来更新到本标准中
