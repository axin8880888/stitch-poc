# 课题010 — 综合图纸解析

## 目标
在一张完整 DXF 上提取所有图元类型，建立空间关系：
- 墙体 → 房间 → 门窗 → 标注 → 家具
- 输出结构化空间描述

## 前置
已完成课题003~009，各自独立解析了：
- 尺寸标注（DIMENSION）
- 图层规范修复
- 布局与视口
- HATCH 填充
- MTEXT 文字
- INSERT 块引用
- 弧形/圆曲线

## 步骤
1. 加载解析 JSON 全量数据
2. 墙体聚合（LINE/LWPOLYLINE → 闭合房间）
3. 门窗定位（ARC + INSERT + 图块）
4. 标注关联（MTEXT → 对应空间）
5. 家具分配（INSERT + CIRCLE）
6. 输出结构化空间 JSON + SVG
