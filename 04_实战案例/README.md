# 第四阶段：项目实战

**目标：建立真实案例库，每个项目完整保存。**

---

## 项目模板

每个项目应包含以下内容：

```
项目名称/
├── 项目信息.md           ← 项目基本信息
├── 原始文件/              ← 现场照片、原始 DWG、手绘草图
├── 输出文件/              ← 最终 DWG、PDF、DXF
├── 修改记录.md            ← 每次修改详细记录
└── 总结.md               ← 项目总结、经验教训
```

## 保存规范

每个项目完成后必须保存：
1. **DWG** - 最终可编辑版本
2. **PDF** - 打印版，按图幅分开
3. **现场照片** - 原始场地照片（至少有参考价值）
4. **最终照片** - 施工完成后的实景照片
5. **修改记录** - 每次修改的详细记录

所有文件按项目归入对应目录。
