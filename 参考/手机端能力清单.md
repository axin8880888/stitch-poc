# 📱 手机端 CAD 能力清单

**机型：** findx9u (aarch64, Android 16)
**主力位置：** ✅ 我们的主战场

---

## 🛠 已有工具

### 核心
| 工具 | 状态 | 用途 |
|------|------|------|
| Python 3.13 | ✅ | 脚本主力 |
| Pillow 12.3 | ✅ | 图像处理、SVG→PNG |
| svgwrite 1.4 | ✅ | SVG 矢量出图 |
| Django? | ❓ | Web 服务（待定） |

### 格式处理
| 能力 | 实现方式 |
|------|---------|
| DXF 解析 | 纯 Python 手写解析器（已验证） |
| DXF 生成 | 纯文本写入（已验证） |
| SVG 绘图 | svgwrite（已验证） |
| SVG→PNG | ImageMagick convert（已验证） |
| PNG 分析 | Pillow（可用） |

### 系统资源
- RAM: 14GB ✅（充足）
- 存储: 248G 可用 ✅（充足）
- CPU: aarch64 8核 ✅

---

## 🔄 核心工作流

```
需求输入（你）
    ↓
OpenClaw 分析 + 生成 DXF
    ↓
输出 SVG 可视化 ← 你手机直接看图
    ↓
若需 PNG 图片 → magick convert
    ↓
修改调整
    ↓
最终交付 DXF + PDF（后续补打印）
```

---

## ❌ 当前没法做的

1. **无法装 CAD 桌面软件**（QCad/LibreCAD 都不行）
2. **无法装 ezdxf 原生版**（缺 numpy 二进制）
3. **没图形界面**（只能命令行转图）
4. **无法直接打印**（PDF 输出需后续扩展）

但这些都不影响我们学习——**重点是用工具辅助理解，而不是依赖工具**。
