## 绘图命令

### LINE（L - 直线）
**用途：** 绘制两点之间的直线段
**参数：** 起点 → 下一点（连续绘制，回车结束）
**场景：** 任何直线元素的绘制
**常见错误：** 应该用 PL（多段线）但用了 L，导致对象不连续
```
命令: L → 指定第一点: 单击 → 指定下一点: 单击 → Enter
```

### PLINE（PL - 多段线）
**用途：** 绘制相连的直线/圆弧段（整体对象）
**参数：** 起点 → 下一点 → A(圆弧)/L(直线)/W(宽度)/C(闭合)
**场景：** 墙体轮廓、封闭区域边界、需要统一宽度的线
**常见错误：** 画闭合区域用 L 逐段画，不如 PL 整体编辑方便
```
命令: PL → 指定起点 → W → 起点宽度50 → 端点宽度50 → 画墙厚
```

### RECTANG（REC - 矩形）
**用途：** 绘制矩形多段线
**参数：** 角点1 → 角点2 / D(尺寸) / R(圆角) / W(宽度)
**场景：** 房间轮廓、门窗洞口、图框
**常见错误：** 用 L 画四条线拼矩形
```
命令: REC → 指定第一角点 → D → 长度 → 宽度 → 单击定位
```

### CIRCLE（C - 圆）
**用途：** 绘制圆形
**参数：** 圆心+半径/直径 / 2P(两点) / 3P(三点) / TTR(切切半)
**场景：** 圆柱、圆孔、圆弧墙体
**常见错误：** 半径与直径混淆
```
命令: C → 指定圆心 → 指定半径: 500
```

### ARC（A - 圆弧）
**用途：** 绘制弧形
**参数：** 三点(默认) / 起点+圆心+端点 / 起点+圆心+角度 等 11 种
**场景：** 弧形墙体、装饰弧线
**常见错误：** 逆时针与顺时针方向混淆
```
命令: A → 指定起点 → C(圆心) → 指定圆心 → 指定端点
```

### ELLIPSE（EL - 椭圆）
**用途：** 绘制椭圆
**参数：** 轴端点+另一轴长 / 圆心+轴端点+旋转
**场景：** 椭圆造型、异形造型
```
命令: EL → 指定轴端点 → 指定另一轴距离
```

## 修改命令

### COPY（CO - 复制）
**用途：** 复制选择的对象
**参数：** 选择对象 → 基点 → 位移点 / M(多重复制) / D(阵列)
**场景：** 复制家具、灯具、标注
**常见错误：** 没有选对基点导致位置偏移
```
命令: CO → 选对象 → 指定基点 → 指定第二点 → (Enter 结束)
```

### MOVE（M - 移动）
**用途：** 移动选择的对象
**参数：** 选择对象 → 基点 → 目标点
**场景：** 位置调整、对齐
**常见错误：** 距离较远时用 M 不如用 STRETCH 或选择过滤器
```
命令: M → 选对象 → 基点 → 目标点
```

### ROTATE（RO - 旋转）
**用途：** 绕基点旋转对象
**参数：** 基点 → 角度 / C(复制) / R(参照旋转)
**场景：** 旋转家具、调整方向
**常见错误：** 正负角度方向混淆（逆时针为正）
```
命令: RO → 选对象 → 基点 → R → 参照角度 → 新角度
```

### SCALE（SC - 缩放）
**用途：** 按比例缩放对象
**参数：** 基点 → 比例因子 / C(复制) / R(参照缩放)
**场景：** 缩放图块、调整比例
**常见错误：** 用 SC 改视口比例（应该用属性面板的视口比例）
```
命令: SC → 选对象 → R → 参照长度 → 新长度
```

### MIRROR（MI - 镜像）
**用途：** 镜像复制选中的对象
**参数：** 选择对象 → 镜像线第一点 → 第二点 → 是否删除源对象(Y/N)
**场景：** 对称户型、对称家具
**常见错误：** 文字也被镜像成了反字（使用 MIRRTEXT=0）
```
命令: MI → 选对象 → 指定镜像线第一点 → 第二点 → N(保留源)
```

### TRIM（TR - 修剪）
**用途：** 以剪切边为界，修剪掉多余线段
**参数：** 选剪切边 → 选要修剪的部分
**场景：** 清理交叉线条、门窗洞口
**常见错误：** 没选到剪切边就按了右键（可以用快速模式"全部剪切"）
```
命令: TR → 空格(全选作为剪切边) → 单击要剪的部分
```

### EXTEND（EX - 延伸）
**用途：** 延长对象到指定的边界
**参数：** 选边界边 → 选要延伸的对象
**场景：** 补齐交线
**常见错误：** 延伸方向不对或边界对象不可见
```
命令: EX → 空格(全选作为边界) → 单击要延伸的部分
```

### OFFSET（O - 偏移）
**用途：** 按指定距离创建平行对象
**参数：** 偏移距离 → 选择对象 → 指定方向 / T(通过)
**场景：** 墙体偏移、轮廓加厚、等高线
**常见错误：** 偏移后不修改图层导致图层混乱
```
命令: O → 100 → 选墙体线 → 点向内方向
```

### FILLET（F - 圆角）
**用途：** 在两条线之间创建圆角
**参数：** R(半径) → 选择第一条线 → 选择第二条线
**场景：** 墙角圆角、轮廓过渡
**常见错误：** 半径过大导致图形变形
```
命令: F → R → 50 → 选第一条线 → 选第二条线
```

### CHAMFER（CHA - 倒角）
**用途：** 在两条线之间创建倒角
**参数：** D(距离) → 第一距离 → 第二距离 → 选线
**场景：** 直角切角
**常见错误：** 两段距离设置反了
```
命令: CHA → D → 100 → 100 → 选第一条线 → 选第二条线
```

### EXPLODE（X - 分解）
**用途：** 将块或多段线分解为单个对象
**参数：** 选择对象
**场景：** 需要编辑块内部或修改多段线分段
**常见错误：** 分解后块属性丢失变为标签
```
命令: X → 选对象 → Enter
```

## 块与插入

### BLOCK（B - 创建块）
**用途：** 将多个对象合并为一个块
**参数：** 名称 → 基点 → 选择对象 → 保留/转换/删除
**场景：** 制作门窗块、家具块、图框
**常见错误：** 基点在远处导致插入时位置偏差
```
命令: B → 输入名称 "DOOR-900" → 选基点(门轴) → 选对象 → OK
```

### INSERT（I - 插入块）
**用途：** 将已定义的块插入到图中
**参数：** 块名 → 插入点 → 比例 → 旋转角度
**场景：** 放置门窗、家具、图框
```
命令: I → 选块名 → 指定插入点 → 确定
```

## 填充与图层

### HATCH（H - 图案填充）
**用途：** 为封闭区域填充图案或颜色
**参数：** 图案类型 → 比例 → 角度 → 选择边界
**场景：** 地面铺装、墙体剖面、吊顶图案
**常见错误：** 边界不封闭导致无法填充
```
命令: H → 选图案(ANSI31/AR-CONC等) → 拾取内部点 → Enter → OK
```

### LAYER（LA - 图层管理器）
**用途：** 创建、管理图层属性
**参数：** 新建/删除/冻结/锁定/颜色/线型/线宽/打印
**场景：** 任何时候都需要管理图层
**常见错误：** 所有对象画在一个图层上
```
命令: LA → 新建 → 命名 A-WALL → 颜色:7 → 线宽:0.30 → OK
```

## 标注与文字

### DIMENSION（D - 标注）
**用途：** 创建尺寸标注
**参数：** 水平/垂直/对齐/角度/半径/直径/坐标
**场景：** 标注墙体长度、间距、半径
**常见错误：** 标注不按实际尺寸（提示：检查是否在布局空间）
```
命令: DLI(线性标注) → 指定第一条延伸线起点 → 第二条 → 放置位置
```

### MTEXT（MT - 多行文字）
**用途：** 输入多段落文字
**参数：** 文字框 → 字体/高度/对齐/行距
**场景：** 施工说明、材料说明、图注
**常见错误：** 文字不按标准字体/字号，在不同机子上显示乱码
```
命令: MT → 拖框 → 输入文字 → 设置字体黑体/高度3.5 → OK
```

### MATCHPROP（MA - 属性匹配）
**用途：** 将一个对象的属性复制到其他对象
**参数：** 源对象 → 目标对象 / S(设置选择匹配属性)
**场景：** 快速统一图层、颜色、线型
**常见错误：** 复制了不需要的属性（用 S 参数筛选）
```
命令: MA → 选源对象 → 选目标 → Enter
```

## 管理与修复

### PURGE（PU - 清理）
**用途：** 删除未使用的图层、块、线型、标注样式等
**参数：** 全部清理 / 逐项选择
**场景：** 图纸交付前清理冗余数据
**常见错误：** 没有勾选"清理嵌套项"导致清理不彻底
```
命令: PU → 勾选全部 → 全部清理 → 反复直到无可清理项
```

### AUDIT（AUDIT - 核查）
**用途：** 检查和修复图形数据错误
**参数：** Y(修复错误) / N(仅报告)
**场景：** 打开异常图纸时
**常见错误：** 忘记执行此步骤，导致后续操作异常
```
命令: AUDIT → Y(修复错误)
```

### RECOVER（RECOVER - 修复）
**用途：** 打开并修复损坏的 DWG 文件
**参数：** 选择文件
**场景：** DWG 文件无法正常打开
```
命令: RECOVER → 选择文件 → 等待修复完成
```

## 打印与输出

### PLOT（PLOT - 打印）
**用途：** 输出图纸到打印机或 PDF
**参数：** 纸张 → 打印机 → 打印范围 → 比例 → 打印样式
**场景：** 每次出图必须用
**常见错误：** 比例不对、图纸偏移、颜色打印不对
```
Ctrl+P → 选择打印机(DWG to PDF) → 图纸 A3 → 打印范围:布局
→ 比例 1:1 → 打印样式 → 确定
```

### LAYOUT（LO - 布局）
**用途：** 创建和管理布局选项卡
**参数：** 新建/删除/重命名/移动/复制
**场景：** 每张图纸一个布局
**常见错误：** 在模型空间排图（应该在布局空间）
```
右键点击布局标签 → 新建布局 → 右键 → 页面设置管理器
```

### PUBLISH（PUBLISH - 批量发布）
**用途：** 将多个布局同时输出为 PDF/DWF/DWFx
**参数：** 选择布局 → 输出格式 → 保存路径
**场景：** 一键输出整套图纸
**常见错误：** 布局中不同图幅混排，需分别设置
```
命令: PUBLISH → 勾选需要输出的布局 → 发布为:PDF
→ 保存路径 → 发布
```
